코드블럭 쓰기

인라인 코드는 백틱 1개로 감싼다.

백틱( ` )은 숫자판 1 왼쪽에 있다. 맥에서는 한/영 바꿔야 쓸 수 있다

코드


`코드`


블럭 코드는 백틱 3개로 감싼다.

("hello, world");

백틱 3개 넣고 언어형태를 적어주면 하이라이트 기능을 준다.
